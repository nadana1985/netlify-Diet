# VEGETABLE PRIORITY (ONLY THESE REPEAT)

Only the following vegetables are allowed for repeated use:

- Sorakkai (Bottle gourd)
- Pudalangai (Snake gourd)
- Parangikkai (Yellow pumpkin)
- Cabbage
- Cauliflower
- Murungakkai (Drumstick)
- Kothavarangai (Cluster beans)

Explicitly disallowed:
- Mixed vegetables
- Generic vegetable curry
- Unnamed vegetables
